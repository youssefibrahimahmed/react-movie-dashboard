# 🎬 React Movie Dashboard

A modern movie dashboard built with React that allows users to browse movies, view detailed information, explore trending movies, and manage authentication.

## 🚀 Features

- 🔐 Login Authentication
- 📊 Dashboard with movie statistics
- 🎬 Film Listing Page
- 🔥 Trending Movies
- 🔍 Search Movies
- 📄 Movie Details Modal
- ⚡ Global State Management using Redux Toolkit
- 📡 Data Fetching with TanStack Query & Axios
- 🛣️ React Router Navigation
- ⏳ Loading States
- ❌ Error Handling
- ♻️ Reusable Components
- 🛡️ Protected Routes
- ✅ Form Validation

## 🛠️ Technologies Used

- React
- Redux Toolkit (RTK)
- TanStack Query
- React Router DOM
- Axios
- Tailwind CSS
- TVMaze API

```bash

npm install

npm run dev
```
